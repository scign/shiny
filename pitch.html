Introduction

There are many times when a set of random numbers from a normal distribution are needed:

  • Test an experimental approach

  • Detect inconsistencies in naturally-occurring data

  • Calculating event probabilities

  • Initialize weights in an AI model

rnorm

The rnorm function in R generates random numbers from the normal distribution, with mean mu and standard deviation sd.

samples <- 100; mu <- 0; sd <- 1
data <- rnorm(samples, mu, sd)
hist(data)

My Shiny app

Using Shiny I provide a graphical front-end to the rnorm function. The user can draw a box in the graph space and generate two sets of random numbers from the normal distribution, and download the data to a CSV file.

The means are the centre of the box and the standard deviations are set such that the bounds of the box are at 2 standard deviations from the mean.

There is also an option in the interface to display contours of the data.
